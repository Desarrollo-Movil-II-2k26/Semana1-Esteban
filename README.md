# REACT NATIVE
¿Qué es React-Native?
• Es un marco de trabajo JavaScript para crear aplicaciones híbridas en plataformas Android o IOS, utilizando la biblioteca React.
• El código de las aplicaciones está escrito en JavaScript y las interfaces de visualización aplican los fundamentos HTML y CSS.
• Aplica los conceptos DOM del desarrollo de páginas web.
• La biblioteca React fue escrita por los equipos de desarrollo de Facebook.
